# 目的地特化装备包 + 极端气候触发器

> 本文件是 SKILL.md 的按需参考。在完成天气数据获取后、生成清单前加载，用于按目的地/气候快速拼装区域套件，而不是每次都靠 WebSearch 现搜。

## 1. 区域套件（Regional Kits）

命中即整卡拼装（每条物品仍走 §0 映射表归属，本表只负责"命中即加"）。任何在其他上游规则中已存在的物品不重复。

### 1.1 美国 US
- 证件类：EVUS 登记打印件（10 年 B1/B2 签证必需）、I-94 电子入境记录截图（打印可选）、儿童出生证明 / 亲子关系证明（未成年入境海关常查）、旅行保险打印件（含急救险）
- 装备类：Type A 转换头、eSIM（AT&T / T-Mobile）、儿童安全座椅（各州 6-8 岁以下强制，自驾必备）、TSA-accepted 密码锁
- 提醒（info-card）：紧急电话 911、中国大使馆 +1-202-495-2216、中国驻洛杉矶总领馆 +1-213-807-8088（南加/内华达）、海关禁带肉类/水果/土壤，现金 > $10,000 须申报

### 1.2 欧洲申根（法/德/意/西/荷/瑞/奥/比等）
- 证件类：申根签证 + 保险 30 万欧最低保额证明打印件、酒店预订全程确认单（申根边检抽查）
- 装备类：Type C/F 转换头、防扒斜挎包（欧洲各大城市扒手警惕）、便携折叠雨衣
- 提醒（info-card）：紧急电话 112、现金 > €10,000 须申报、禁带肉/奶制品

### 1.3 日本 JP
- 证件类：Visit Japan Web QR 码（推荐提前登记）、酒店预订确认单
- 装备类：Type A 转换头、Suica/PASMO IC 卡、口罩（花粉季 3-5 月必备）、现金 ¥50,000 起步（部分地方仍不刷卡）
- 提醒（info-card）：紧急电话 110/119、中国大使馆 +81-3-3403-3388、禁带肉/水果/种子

### 1.4 泰国 TH
- 证件类：TDAC 电子入境卡（2025 年起强制）、酒店预订确认单
- 装备类：Type A/C 混合转换头、防蚊液（DEET ≥ 20%）、腹泻药（诺氟沙星/蒙脱石散）、防晒 SPF50+
- 提醒（info-card）：紧急电话 191/1155（旅游警察）、现金 < 20,000 泰铢无需申报、电子烟入境违法

### 1.5 热带 + 伊斯兰文化圈（东南亚 / 中东 / 北非 / 南亚共通）

用于目的地不在 1.1-1.4 单独列出、但属于热带 / 穆斯林文化区域的行程。命中即在前面 1.3-1.4 已列条目基础上追加：

- 装备类：万能转换头（各国插座混杂）、腹泻药 / 蒙脱石散 / 口服补液盐、清凉油 / 风油精、宽檐帽（防晒）、可湿性大水壶
- 服饰类（伊斯兰文化圈：阿联酋 / 沙特 / 埃及 / 印尼等穆斯林占多数地区）：女性长袖 + 长裤 + 披肩（进清真寺遮头必需）、男性避免短裤 tank top
- 提醒（info-card）：多数国家禁带猪肉制品 / 酒精饮料（非穆斯林可少量）、公共场所斋月期间日间禁食、目的地紧急电话（现查现写）

### 1.6 澳新 AU/NZ
- 证件类：ETA 电子签或 eVisitor 打印件、儿童亲子关系证明（未成年）
- 装备类：Type I 转换头、强效防晒 SPF50+（澳新紫外线全球最强）、宽檐帽、驱蚊液
- 提醒（info-card）：全球最严检疫，禁带肉/蛋/奶/种子/水果/中药材，违者罚款 A$5,500+；现金 > A$10,000 须申报

### 1.7 高原地区（西藏 / 川西 / 云贵高原 / 秘鲁高地 / 玻利维亚 / 尼泊尔）
- 药品类：红景天（提前 1 周服用）、高原安、便携氧气瓶（当地租/带）、葡萄糖
- 装备类：SPF50+ 防晒霜、润唇膏、护手霜、保温壶、头灯、防风外套（早晚温差 > 15°C）
- 提醒（info-card）：高反症状识别、最近高压氧舱位置、避免感冒 / 洗澡受凉

### 1.8 东北亚寒地（哈尔滨 / 长白山 / 首尔冬季 / 北海道）
- 装备类：抓绒内胆、羽绒服（≥ 800 蓬松度）、防风保暖裤、羊毛袜 3 双、暖宝宝、防雾镜片保暖眼镜、镜面/防滑鞋
- 提醒：极寒下手机会关机，充电宝须贴身保暖

## 2. 极端气候触发器（Weather Triggers）

按行程期间任一日的天气数据自动触发，与区域套件叠加。

### 2.1 高温 ≥ 38°C（如 Vegas / 迪拜 / 印度夏季 / 撒哈拉）
必加卡：**沙漠/高温装备**
- 1L 保温水壶 ×1、电解质补充剂（宝矿力/佳得乐粉包 5+）、冰袖、宽檐遮阳帽、遮阳伞、湿巾降温
- 提醒：正午避免外出，注意热射病症状

### 2.2 极寒 ≤ 0°C
必加卡：**防寒装备**
- 保暖内衣 ×2 套、羊毛袜 ×3、防风羽绒服、抓绒中层、保暖手套、围巾/毛帽、暖宝宝 10+ 片、护脸口罩
- 提醒：手机贴身放，充电宝低温掉电快

### 2.3 紫外线指数 ≥ 10（高原 / 澳洲 / 中东 / 热带海岛）
必加：SPF50+ 防晒霜、SPF30+ 润唇膏、UV400 墨镜、防晒衣（UPF ≥ 50）、宽檐帽
- 提醒：晒后修复芦荟胶必带；儿童须专用儿童防晒（低 SPF/物理防晒）

### 2.4 温差 ≥ 15°C（早晚 / 跨城 / 高原昼夜）
必加：可拆卸洋葱式分层（内层排汗速干 + 中层薄抓绒 + 外层防风）
- 穿搭建议区块强制标注"洋葱式分层"

### 2.5 强降水 / 雨季 / 台风季（东南亚 6-9 月 / 华南梅雨 / 日本梅雨）
必加：便携雨衣（比雨伞更适合手拿相机/带娃场景）、防水鞋套或速干鞋、防水袋（护电子设备）、备用衣物 +2 套
- 提醒：地质灾害预警、避险 App（如日本 NHK / 中国国家应急广播）

### 2.6 花粉/沙尘（春季日韩 / 华北沙尘暴 / 中亚沙尘）
必加：N95 口罩 ×5、抗过敏药（氯雷他定/西替利嗪）、鼻腔喷雾（生理盐水）、护目镜（重度沙尘）

## 3. 使用规则

1. **叠加而非替换**：区域套件 + 气候触发器可同时命中（如 Vegas 7 月 = US 套件 + 高温 + 强紫外线）。合并去重后按 §0 映射表归属分类。
2. **触发前置**：任何区域/气候触发都必须在 §0 映射表判定之前发生——先决定"要加什么"，再决定"归哪张卡"。
3. **未覆盖国家**：不在本表中的国家继续走 SKILL.md 的 WebSearch 兜底流程（§7 packing-rules 未覆盖国家兜底），本表不阻塞。
4. **合规时效**：入境政策/签证要求容易过期。所有 info-card 中的条目必须追加水印"信息截至 2026 年上半年，出发前请至驻外使领馆官网核查"，并在 T-30 天临走待办中提示复核。
